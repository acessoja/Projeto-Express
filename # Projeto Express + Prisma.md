# Projeto Express + Prisma
**Linguagem:** Node.js
**ORM:** Prisma
**Dependências:** `express`, `@prisma/client`, `prisma`

## Como rodar
1. Instale as dependências: `npm install`
2. Gere o banco de dados: `npx prisma db push`
3. Execute o servidor: `node index.js`
4. A API estará rodando em `http://localhost:3000/usuarios`
